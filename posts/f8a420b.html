一款好用去除广告的浏览器插件 uBlock Origin

Created2025-06-20|Updated2026-01-26|插件
|Post Views:

今天给大家介绍这一款适合很多人的浏览器插件 —— ​uBlock Origin​。无论你是看 Bilibili / YouTube 等平台的视频，还是浏览普通网页，uBlock Origin 都能帮你自动跳过广告，为你带来更清爽的使用体验。

除了视频方面，uBlock Origin 在打开新闻网站、电商页面、论坛等网页时，也能有效屏蔽弹窗广告、横幅广告、追踪脚本等烦人的内容。虽然它会在运行时占用一定的内存和 CPU 资源，但整体使用起来非常流畅，几乎感觉不到卡顿。确实是一款“装了就离不开”的神器。

🌟 uBlock Origin 的主要优点

  • ​完全免费、开源​：不像其他某些插件会暗藏收费机制，uBlock Origin 完全开源，安全放心。
  • ​占用资源小​：相比一些老牌的广告插件，uBlock Origin 更加轻量高效。
  • ​规则更新及时​：内置 EasyList、uBlock filters 等规则库，同时支持手动添加自定义过滤规则。
  • ​高度可配置​：高级用户可以根据需要自定义屏蔽范围和行为。

🚀 如何安装 uBlock Origin

  1. 打开你的浏览器插件商店：

    • Chrome 用户进入 Chrome 网上应用店
    • Firefox 用户访问 Firefox Add-ons
    • Edge 用户进入 Edge 插件商店

  2. 搜索 uBlock Origin 并点击“添加到浏览器”。

  3. 安装完成后，在右上角扩展图标中点击即可进行简单配置。

  4. 如果拓展商店没法搜索到或者无法使用可以在github页面下载对应的浏览器拓展程序

    • Github uBlock Origin页面Github uBlock Origin页面下载

⚙️ 推荐设置

为了获得更强的屏蔽效果，可以进入插件设置页面，勾选以下选项：

  • ✔️ 启用“uBlock filters – Privacy”
  • ✔️ 启用“uBlock filters – Annoyances”
  • ✔️ 启用“AdGuard Chinese filter”（适合中文网页）
  • ✔️ 启用“Fanboy’s Annoyance List”

如果你愿意深入研究，还可以添加自定义的规则，如屏蔽某些弹窗提示、视频站点的特定广告行为等。

🕹️如何使用

  • 安装好的插件你需要固定好插件按照我一下图片操作


  • 这样你就已经开启了拦截了，有些网页还是会出现广告没有拦截到要怎么搞呢？我们可以用手动临时移除元素来拦截。

  • 点击你想要拦截的广告，我以我之前的英文网站为例子就拦截图片


  • Youtube另外可以拦截视频广告，真的很方便

❗ 注意事项

虽然 uBlock Origin 很强大，但也可能会误伤某些正常内容，比如网站评论区、内嵌视频播放器等功能。因此如果你遇到页面功能不正常的情况，可以临时点击扩展图标，选择“在此网站禁用”，刷新页面即可。

✅ 总结

如果你厌倦了网页上各种广告弹窗，或者不想在视频开始前强制观看几十秒的广告，那么 uBlock Origin 是你不容错过的插件。它简单、干净、强大，装上之后，你可能再也不想卸载它了。

快去安装试试吧！让你的浏览器焕然一新 🎉
