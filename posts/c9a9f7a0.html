深圳龙华追缴公积金全流程（少缴未缴必看）

Created2026-07-20|Updated2026-07-20|公积金维权生活记录
|Post Views:

📝 一、预约流程（关键！号源紧张）

操作平台：微信关注深圳市住房和建设局公众号

具体路径：业务办理 → 公积金预约 → 预约申请

核心填写要点：

  • 业务名称：选择【投诉单位少缴未缴】

  • 预约网点：选择单位公积金开户地或经营所在地（龙华办理选对应网点即可）

  • 信息填写：单位名称务必填公司全称，手机号填自己的（用于接收验证码，务必准确）

⚠️ 重要提醒：号源非常紧张，经常需要抢号，建议提前规划时间，定时蹲守预约入口。

📂 二、材料准备（建议提前打印，避免现场耽误时间）

以下材料按需准备，在职和离职人员略有区别，建议全部打印纸质版，复印件也可提前准备好备用：

  1. 本人身份证（原件+复印件，复印件建议提前复印好）

  2. 劳动合同（在职人员必备，证明劳动关系）

  3. 离职证明（已离职人员必备，在职人员无需准备）

  4. 银行工资流水（核心材料，仅需工资相关流水即可）

  5. 纳税记录（证明收入情况，辅助核算公积金缴存基数）

  6. 社保缴费记录（佐证劳动关系和在职时间）

📌 材料打印小贴士（手把手教操作）：

💳 银行工资流水

两种方式均可：① 登录对应银行 APP，搜索“交易流水打印”，选择工资流水区间，申请电子版后自行打印；② 携带身份证到银行柜台，让工作人员帮忙打印，全程免费。

🧾 纳税记录

操作路径：登录深圳电子税务局 → 特色应用 → 纳税记录开具 → 选择对应时间（建议选全年，覆盖缴存争议时段） → 生成 PDF 后下载打印，打印出来的记录会有电子签章，无需额外盖章。

🏢 社保缴费记录

操作路径：广东政务服务网 → 个人登录（实名认证） → 搜索“个人权益记录查询打印” → 选择深圳地区 → 下载参保证明后打印。

⚠️ 补充提醒：现场也可打印社保和纳税记录，但现场使用麒麟系统，不熟悉操作的话会耽误时间，建议提前在家准备好，办理效率更高。

🏢 三、现场办理（龙华点位专属指引）

按照预约时间前往网点，全程流程简单，耗时约30分钟，具体步骤如下：

  1. 到达预约网点后，关注深圳公积金公众号，在公众号内完成预约取号（凭预约信息取号，避免白跑）；

  2. 叫号后，到对应窗口提交准备好的全部材料，工作人员会核对信息；

  3. 核对无误后，签署相关投诉表格，确认投诉信息，完成提交即可。

📍 龙华办理专属指引：乘坐地铁到深圳北站，出站后导航至对应公积金网点，距离很近，步行即可到达，建议提前10分钟到网点，避免错过预约时间。

⏳ 四、后续流程（关键！别忘个人补缴）

材料提交后，无需额外操作，耐心等待公积金中心联系公司，责令公司补缴欠缴的公积金即可。

⚠️ 核心提醒：公司补缴完成后，个人部分也需要补缴，具体操作路径如下：

深圳公积金公众号 → 我要查询 → 办事指南 → 缴存类 → 个人补缴，按照系统提示完成个人部分的补缴操作即可。

✅ 全流程总结（划重点）

  • ✔ 提前预约：号源紧张，务必提前抢号，避免跑空

  • ✔ 材料齐全：在职/离职材料区分准备，提前打印好，一次带齐

  • ✔ 办理高效：现场流程简单，全程约30分钟，按步骤操作即可

  • ✔ 个人补缴：公司补缴后，及时完成个人部分补缴，避免影响公积金使用

龙华实测有效，亲测一次办理成功，打工人遇到公积金欠缴、少缴问题，别忍气吞声，按照流程操作即可维权～
