域名注册与 Cloudflare 托管教程

本指南将引导你完成域名注册，以及使用 Cloudflare 进行域名托管的步骤。

注册域名

推荐服务商：NameSilo

我们推荐使用 NameSilo 进行域名注册，它操作简便且价格合理。点击这里 直接访问 NameSilo 官网。

域名后缀选择

  • .com： 推荐用于正式网站，更具专业性。
  • .xyz / .site： 适合个人博客、学生项目等，价格相对较低。

请根据你的需求选择合适的域名后缀。

NameSilo 用户注册

注册注意事项

  • 邮箱： 请使用你常用的有效邮箱进行注册，以便接收重要通知。
  • 用户名： 注册时设置的用户名请务必牢记。

填写个人信息

  1. 注册完成后，你需要填写个人信息。
  2. 访问 美国地址生成器 或直接复制此网址：https://www.meiguodizhi.com。
  3. 在 NameSilo 网站，点击 Account Settings，按照生成器提供的身份信息填写。

配置 DNS 服务器

  1. 填写完个人信息后，点击 Dashboard。
  2. 如果你已购买域名，会看到 Active Domains，点击 Manage。
  3. 点击你的域名，找到 nameserver，点击 Edit。
  4. 删除所有已有的 nameserver，并保持此页面打开，我们稍后会用到。

Cloudflare 托管域名

添加域名到 Cloudflare

  1. 访问 Cloudflare 官网，点击 添加域。
  2. 输入你购买的域名，例如 123.xyz。
  3. 选择 免费计划 (如果你有特殊需求，也可以考虑其他付费计划)。

配置 NameSilo DNS 服务器

  1. 返回到之前在 NameSilo 保留的 nameserver 编辑页面。
  2. 将 Cloudflare 提供的 DNS 服务器地址粘贴到 NameSilo 的 nameserver 中。
  3. 保存更改。

检查 Cloudflare 配置

  1. 返回 Cloudflare 页面，点击 检查 按钮。
  2. 通常情况下，配置会在 5 分钟内生效。
  3. 如果配置未生效，请稍后或隔天再进行检查。

完成

一旦配置生效，你就可以使用你的域名了。
